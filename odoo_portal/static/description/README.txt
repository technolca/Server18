MODULE DESCRIPTION IMAGES
=========================

This folder contains images used for module description and display in the Apps list.

Files:
- icon.png (128x128 pixels recommended) - Module icon shown in Apps list
- banner.png or banner.jpg - Optional banner image for module description

Image Specifications:
- Format: PNG (preferred) or JPG
- Icon Size: 128x128 pixels (square)
- Banner Size: 1920x300 pixels (recommended)
- Transparent background preferred for icons
