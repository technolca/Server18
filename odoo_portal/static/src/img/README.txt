MODULE IMAGES
=============

This folder contains images used within the module (templates, views, etc.).

Best Practices:
- Use descriptive file names (e.g., employee_portal_logo.png)
- Keep images optimized for web (reasonable file sizes)
- Use PNG for images with transparency
- Use JPG for photos or images without transparency

Image Usage in Templates:
- Reference images using: /odoo_portal/static/src/img/your_image.png
- Or use: <img t-att-src="'/odoo_portal/static/src/img/your_image.png'"/>
